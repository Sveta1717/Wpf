ORM, DAL, BLL...
Хранение данных в БД популярно, однако создает определенные проблемы при работе с
типичными языками программирования (ЯП). В ЯП нет типов данных, подходящих для
"таблиц", а в БД таблица - основной вид данных.

Наиболее современный подход заключается в создании объектов. описывающих строки
(шапки) таблиц и коллекций таких объектов. представляющих данные из БД.

                  ORM
DB                 =>   Lang                             DAL
TABLE Department        class Department {               Data Access Layer
  Id                        Id,                          Инструменты, которые переведут
  Name                      Name }                 ORM   работу с коллекцией в запросы
                        List<Department> Department =>   Departments.Find("IT Отдел")

BLL  - Buiseness Logic Layer - "слой" программы, обрабатывающий данные
   (основные программы)
